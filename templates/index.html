Mathematical Framework

The Gauss-Seidel method is an iterative technique utilized to solve linear systems in the form $$ Ax = b $$. Unlike the Jacobi method, which computes all new values using only the previous iteration's data, Gauss-Seidel utilizes the most recently updated values immediately. This "forward substitution" approach generally results in a significantly faster rate of convergence.

Worked Example 1

$ 4x_1 - x_2 = 3 $
$ -x_1 + 4x_2 - x_3 = 5 $
$ -x_2 + 4x_3 = 4 $

We isolate each variable, preparing our iterative equations:

$ x_1 = \frac{3 + x_2}{4} $

$ x_2 = \frac{5 + x_1 + x_3}{4} $

$ x_3 = \frac{4 + x_2}{4} $

Using an initial guess of $x^{(0)} = [0, 0, 0]$, we compute Iteration 1:

  • $ x_1 = \frac{3 + 0}{4} = 0.75 $
  • $ x_2 = \frac{5 + 0.75 + 0}{4} = 1.4375 $
  • $ x_3 = \frac{4 + 1.4375}{4} = 1.359375 $
