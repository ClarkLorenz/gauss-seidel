Mathematical Framework

The Gauss-Seidel method is an iterative technique utilized to solve linear systems in the form \( Ax = b \). Unlike the Jacobi method, which computes all new values using only the previous iteration's data, Gauss-Seidel utilizes the most recently updated values immediately within the same loop. This "forward substitution" approach generally results in a significantly faster rate of convergence.

Structured Algorithm
01
Formulate the system \(Ax = b\) and ensure there are no zeros on the main diagonal.
02
Isolate each variable \( x_i \) on the left side of its respective equation.
03
Provide an initial guess vector, typically \( x^{(0)} = [0, 0, \dots, 0] \).
04
Calculate new values by immediately substituting the newest available results for previous variables.
05
Repeat until the error tolerance \( \varepsilon \) is met.
Error Analysis & Convergence

After \( k \) iterations, the absolute error is checked using the maximum difference between the new and old vectors:

\( \max |x_i^{(k+1)} - x_i^{(k)}| < \varepsilon \)

For convergence to be guaranteed regardless of the initial guess, the matrix \( A \) must be strictly diagonally dominant:

\( |a_{ii}| > \sum_{j \neq i} |a_{ij}| \quad \text{for all } i \)
Advantages & Limitations

✓ Advantages

  • Generally converges twice as fast as the Jacobi method.
  • Requires less memory as variables are updated in-place.
  • Easy algorithmic implementation.

✗ Limitations

  • Requires diagonal dominance or positive-definiteness to guarantee convergence.
  • Harder to parallelize than Jacobi due to sequential variable dependence.

Worked Example 1

\( 4x_1 - x_2 = 3 \)
\( -x_1 + 4x_2 - x_3 = 5 \)
\( -x_2 + 4x_3 = 4 \)

Step 0: Isolate variables to prepare iterative equations

\( x_1 = \frac{3 + x_2}{4} \)

\( x_2 = \frac{5 + x_1 + x_3}{4} \)

\( x_3 = \frac{4 + x_2}{4} \)

Using an initial guess of \( x^{(0)} = [0, 0, 0] \), we begin iterating:

Mathematical Discussion

The Gauss-Seidel Method is an iterative technique for solving a square system of \(n\) linear equations with unknown variables \( Ax = b \). It is a modification of the Jacobi method, designed to accelerate convergence.

Core Concept: Unlike the Jacobi method, which requires the retention of the entire vector \( x^{(k)} \) until the entire new vector \( x^{(k+1)} \) is computed, Gauss-Seidel uses the newly computed \( x_i^{(k+1)} \) immediately in the evaluation of the subsequent elements within the exact same iteration loop.
Algorithm
01
Choose an initial guess vector \( x^{(0)} \) and a tolerance level \( \varepsilon \).
02
Isolate each variable \( x_i \) to form the iteration equations.
03
For each \( i = 1, 2, \dots, n \), compute the new value using the most recently updated variables: $$ x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)} \right) $$
04
Check convergence: If \( \max |x_i^{(k+1)} - x_i^{(k)}| < \varepsilon \), the system has converged. Stop.
05
Otherwise, set \( k = k + 1 \) and return to Step 03.
Convergence Requirements

The method is not guaranteed to converge for all matrices. However, convergence is strictly guaranteed if the matrix \( A \) is either symmetric positive-definite or strictly diagonally dominant:

$$ |a_{ii}| > \sum_{j \neq i} |a_{ij}| \quad \text{for all } i $$
Advantages & Limitations

✓ Advantages

  • Converges faster than the Jacobi method in most scenarios.
  • Highly memory efficient (requires only one vector \(x\) since updates happen in-place).
  • Simple algorithmic implementation.

✗ Limitations

  • Requires specific matrix properties (diagonally dominant) to guarantee convergence.
  • Sequential updates make it difficult to parallelize on modern multi-core processors.
EXAMPLE 1 Solve the linear system
\( 4x_1 - x_2 = 3 \)
\( -x_1 + 4x_2 - x_3 = 5 \)
\( -x_2 + 4x_3 = 4 \)

Step 0: Verify and Formulate

The matrix is strictly diagonally dominant (\( 4 > |-1|, 4 > |-1| + |-1|, 4 > |-1| \)). We isolate each variable:

\( x_1 = \frac{1}{4} (3 + x_2) \)
\( x_2 = \frac{1}{4} (5 + x_1 + x_3) \)
\( x_3 = \frac{1}{4} (4 + x_2) \)

Initial guess: \( x^{(0)} = [0, 0, 0] \).

k x₁ x₂ x₃ Status
1 0.75000 1.43750 1.35938 Iterating
2 1.10938 1.86719 1.46680 Iterating
3 1.21680 1.92090 1.48022 Iterating
... ... ... ... ...
~15 1.33333 2.33333 1.58333 Converged ✓
Result: After sufficient iterations, the vector converges to $$ x \approx [1.3333, 2.3333, 1.5833] $$

Verification:

\( 4(1.333) - 2.333 \approx 3 \) ✓

EXAMPLE 2 Solve the linear system
\( 5x_1 + x_2 + 2x_3 = 10 \)
\( x_1 + 4x_2 - x_3 = 5 \)
\( 2x_1 - x_2 + 5x_3 = 12 \)

Step 0: Formulate Iterative Equations

Isolating \( x_1, x_2, \) and \( x_3 \) to the left side yields:

\( x_1 = \frac{1}{5} (10 - x_2 - 2x_3) \)
\( x_2 = \frac{1}{4} (5 - x_1 + x_3) \)
\( x_3 = \frac{1}{5} (12 - 2x_1 + x_2) \)

Initial guess: \( x^{(0)} = [0, 0, 0] \).

k x₁ x₂ x₃ Status
1 2.00000 0.75000 1.75000 Iterating
2 1.15000 1.40000 2.22000 Iterating
