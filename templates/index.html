Mathematical Discussion

The Gauss-Seidel Method is an iterative technique for solving a square system of \(n\) linear equations with unknown variables \( Ax = b \). It is a modification of the Jacobi method, designed to accelerate convergence.

Core Concept: Unlike the Jacobi method, which requires the retention of the entire vector \( x^{(k)} \) until the entire new vector \( x^{(k+1)} \) is computed, Gauss-Seidel uses the newly computed \( x_i^{(k+1)} \) immediately in the evaluation of the subsequent elements within the exact same iteration loop.
Algorithm
01
Choose an initial guess vector \( x^{(0)} \) and a tolerance level \( \varepsilon \).
02
Isolate each variable \( x_i \) to form the iteration equations.
03
For each \( i = 1, 2, \dots, n \), compute the new value using the most recently updated variables: $$ x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j=1}^{i-1} a_{ij} x_j^{(k+1)} - \sum_{j=i+1}^{n} a_{ij} x_j^{(k)} \right) $$
04
Check convergence: If \( \max |x_i^{(k+1)} - x_i^{(k)}| < \varepsilon \), the system has converged. Stop.
05
Otherwise, set \( k = k + 1 \) and return to Step 03.
Convergence Requirements

The method is not guaranteed to converge for all matrices. However, convergence is strictly guaranteed if the matrix \( A \) is either symmetric positive-definite or strictly diagonally dominant:

$$ |a_{ii}| > \sum_{j \neq i} |a_{ij}| \quad \text{for all } i $$
Advantages & Limitations

✓ Advantages

  • Converges faster than the Jacobi method in most scenarios.
  • Highly memory efficient (requires only one vector \(x\) since updates happen in-place).
  • Simple algorithmic implementation.

✗ Limitations

  • Requires specific matrix properties (diagonally dominant) to guarantee convergence.
  • Sequential updates make it difficult to parallelize on modern multi-core processors.
EXAMPLE 1 Solve the linear system
\( 4x_1 - x_2 = 3 \)
\( -x_1 + 4x_2 - x_3 = 5 \)
\( -x_2 + 4x_3 = 4 \)

Step 0: Verify and Formulate

The matrix is strictly diagonally dominant (\( 4 > |-1|, 4 > |-1| + |-1|, 4 > |-1| \)). We isolate each variable:

\( x_1 = \frac{1}{4} (3 + x_2) \)
\( x_2 = \frac{1}{4} (5 + x_1 + x_3) \)
\( x_3 = \frac{1}{4} (4 + x_2) \)

Initial guess: \( x^{(0)} = [0, 0, 0] \).

EXAMPLE 2 Solve the linear system
\( 5x_1 + x_2 + 2x_3 = 10 \)
\( x_1 + 4x_2 - x_3 = 5 \)
\( 2x_1 - x_2 + 5x_3 = 12 \)

Step 0: Formulate Iterative Equations

Isolating \( x_1, x_2, \) and \( x_3 \) to the left side yields:

\( x_1 = \frac{1}{5} (10 - x_2 - 2x_3) \)
\( x_2 = \frac{1}{4} (5 - x_1 + x_3) \)
\( x_3 = \frac{1}{5} (12 - 2x_1 + x_2) \)

Initial guess: \( x^{(0)} = [0, 0, 0] \).
